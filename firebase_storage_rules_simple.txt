Firebase Storage 간단한 테스트용 규칙
==========================================

개발 중 테스트를 위한 간단한 규칙:

rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /{allPaths=**} {
      // 인증된 사용자는 모든 파일 읽기/쓰기 가능
      allow read, write: if request.auth != null;
    }
  }
}

주의: 이 규칙은 개발/테스트용입니다. 프로덕션에서는 더 엄격한 규칙을 사용하세요.
